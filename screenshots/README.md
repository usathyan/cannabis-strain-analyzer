# Screenshots Directory

This directory contains screenshots of the Cannabis Strain Analysis & Comparison application.

## Files to Add:

1. **configuration-tab.png** - Screenshot of the Configuration tab showing:
   - Profile management interface
   - Current profile summary with cannabinoids and terpenes
   - Strain selection grid with "blue dream" and "Blueberry Thai" selected
   - Add custom strain functionality
   - Profile management buttons

2. **compare-tab.png** - Screenshot of the Compare tab showing:
   - Strain analysis interface
   - Z-Score comparison checkbox enabled
   - "Sour Zkittlez" analysis results
   - Similarity score showing "-49.1% Similarity"
   - AI analysis and recommendations

## How to Add Screenshots:

1. Take screenshots of the application running at `http://localhost:8000`
2. Save them as `configuration-tab.png` and `compare-tab.png`
3. Place them in this `screenshots/` directory
4. The README.md will automatically display them

## Screenshot Guidelines:

- Use PNG format for best quality
- Capture the full browser window
- Ensure the interface is clearly visible
- Show the application in action with real data
